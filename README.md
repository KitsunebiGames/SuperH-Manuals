# SuperH Manuals

This repository contains a collection of manuals for the SuperH architecture.

The manuals are split into categories based on what part of the SuperH architecture they apply to.

## Categories

| Category    | Description                                         |
|-------------|-----------------------------------------------------|
| `ISA/`      | CPU Architecture Documentation                      |
| `ABI/`      | Higher level interfaces for C, etc.                 |
| `Hardware/` | Manuals made for specific SuperH Hardware revisions |
